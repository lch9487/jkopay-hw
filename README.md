## Getting Started

First, run the development server:

```bash
npm install

npm run start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Learn More

- Use `npx create-react-app` and `typescript`.
- Use `react-query` to fetch and mutate data.
- Use `mock service worker` to mock data.
- Use `chakra-ui` for fast deveopment.
- Use `navigator.onLine` to detect whether user is online.
